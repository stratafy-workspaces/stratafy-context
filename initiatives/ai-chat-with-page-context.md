## Overview

Embedded AI chat panel available on any workspace page, pre-loaded with relevant page context. Enables natural language interaction with strategic data — asking questions, editing financial models, and exploring scenarios.

## Scope

- Chat panel embedded across 9+ workspace pages with fullscreen mode
- Automatic context loading: current entity, related data, and strategic hierarchy
- Natural language commands for data manipulation (e.g., financial model editing)
- Conversational exploration of strategic scenarios and what-if analysis
- Integration with MCP tools for read/write operations on workspace data

## Expected Outcomes

- Reduce friction between strategic thinking and data entry to near zero
- Enable non-technical users to query and modify complex strategic data conversationally
- Create the primary interaction surface for the Strategic Advisor workflow
- Establish the pattern for AI-assisted decision-making across all product surfaces

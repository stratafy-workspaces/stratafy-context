# Agent Registry & Context Packaging

## Description
Agent Registry & Context Packaging unifies AI agent registration and role-based context delivery, with auto-refresh and management tools for efficient operations.

Context generation serves four resolution levels:
- **Full resolution**: Complete strategic architecture — every layer, every object, every linkage. Used for perspective reviews, alignment scans, strategic planning. Most context-intensive pattern.
- **Strategic resolution**: Foundation, active strategies, key objectives, critical assumptions, top risks. Used for leadership decisions, investor conversations, board updates, strategic advisor engagements.
- **Operational resolution**: Specific strategy, initiative, objectives, and metrics relevant to a function or team. Used for department planning, initiative scoping, weekly execution.
- **Minimal resolution**: Core values, mission, current priorities. Used for content generators, customer-facing bots, routine automation. Enough for organisational identity alignment without strategic depth.

Context profiles for internal roles define which resolution level and which architectural slices each role receives. External context profiles (for advisors, legal, investors) are handled by the External Context Packaging initiative but use the same resolution framework.

Resolution is not a quality judgment — minimal is not inferior to full. It is the right amount of context for the consumer that needs it. The discipline is matching resolution to need.

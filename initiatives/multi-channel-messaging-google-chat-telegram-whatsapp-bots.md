## Overview

Strategic intelligence delivered through messaging platforms where leaders already operate. Each channel surfaces strategic context outside the main application, ensuring intelligence reaches decision-makers without requiring app engagement.

## Scope

### Google Chat Bot
- AI-powered responses using Grok model
- Linear integration for task context
- Daily summary cron for workspace highlights

### Telegram Adapter
- External accounts for cross-platform identity
- Strategic context delivery and query responses

### WhatsApp CEO Morning Briefing
- Daily curated strategic briefing via WhatsApp Business API
- Synthesized workspace state in concise, actionable format

## Strategic Context
The previous Chat SDK strategy was dissolved, but multi-channel delivery continues as a key surface under Strategic Delivery. The focus is on intelligence delivery, not building a chat platform.

## Expected Outcomes
- Leaders receive strategic intelligence in their preferred messaging platform
- Daily briefings reduce time-to-awareness for strategic changes
- Cross-platform identity enables unified experience across channels

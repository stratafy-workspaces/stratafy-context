# AI Tools Directory

## Description
Curated directory of AI tools and agents for business, positioning Stratafy as the governance layer. Dual purpose: SEO/content asset plus market research for integration opportunities.

UPDATED ANGLE (reflecting Jan 29 clarity):

Each tool review now highlights the four problems:
1. Context Problem - Does this tool know your strategy?
2. Visibility Problem - Can you see what it is doing?
3. Freshness Problem - Is it working with current context?
4. Guardrails Problem - What constraints exist?

This positions every AI tool as needing Stratafy as the control plane.

PHASE 1 TOOLS (Q1 2026):
- ClawdBot (published) - Autonomous agent, no strategic context
- Claude Desktop - Powerful but needs governance for org use
- Claude Code - Great execution, who governs what it builds?
- Lindy.ai - Automates workflows, lacks organizational intent

TARGET: 4 tools published by Feb 15, 2026

CONTENT APPROACH:

Each review includes:
1. Overview - What the tool does
2. Business Use Cases - Who its for
3. Strengths - What it does well
4. The Four Problems - Where it lacks (the Stratafy angle)
5. Control Plane Fit - How Stratafy provides governance
6. Pricing

Phase 2 (Q2+): Maintenance mode, 1 review per month max. Prioritize MCP-compatible tools.

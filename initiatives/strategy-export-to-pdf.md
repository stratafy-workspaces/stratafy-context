## Overview

Build a tool that pulls the full workspace strategy (via get_workspace_snapshot) and exports it as a well-formed PDF document. Core value proposition for advisory firms — generate professional strategy documents from live workspace data.

## Scope

### Output Formats
- **Strategy-on-a-Page**: One-pager for client presentations
- **Comprehensive Strategic Operating Document**: Board pack style, full strategy tree
- **Alignment Report**: Scored analysis with gaps and recommendations

### Implementation Options
- MCP tool for AI-driven generation
- API endpoint for programmatic access
- UI feature for self-service export

### Content Sources
- Full workspace snapshot data
- Strategy tree with all components
- Metrics, objectives, and initiative status
- Risk register and assumption state

## Expected Outcomes
- Advisory firms generate professional deliverables from live data
- Board presentations created in minutes, not hours
- Strategy documents always reflect current workspace state

## Overview

Product capability to generate permissioned, curated strategic context packages for external consumers. Each external party receives a role-appropriate slice of the strategic architecture, structured and current, replacing manual briefings with always-current views.

## Scope

### External Access Profiles
- Inclusion/exclusion rules per external party
- Resolution levels and sensitivity boundaries
- Standard profiles for common roles: auditor, legal counsel, strategic advisor, potential partner, investor in diligence

### Governance Layer
- Organisation controls what is exposed
- Defaults to minimum necessary context
- Trust levels per relationship determining additional context

### Always-Current Property
- When strategy updates, external views update automatically
- No re-briefing required after changes

## Strategic Value
- GTM enabler: advisory firms receive structured context from clients, making Stratafy indispensable to the advisory relationship
- Strengthens consumption-as-maintenance flywheel — external consumers stress-test architecture
- Supports the "context layer becomes a network" thesis: value increases with number and diversity of consumers

## Expected Outcomes
- External parties receive professional, current strategic context without manual effort
- Advisory relationships deepened through structured intelligence sharing
- Network effects from diverse external consumption patterns

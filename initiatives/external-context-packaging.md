# External Context Packaging

## Description
Product capability to generate permissioned, curated strategic context packages for external consumers — lawyers, accountants, advisors, investors, partners, and acquirers. Each external party receives a role-appropriate slice of the strategic architecture, structured and current, replacing manual briefings.

Key components:
- External access profiles with inclusion/exclusion rules, resolution levels, and sensitivity boundaries
- Standard profiles for common external roles (auditor, legal counsel, strategic advisor, potential partner, investor in diligence)
- Governance layer: organisation controls what is exposed, defaults to minimum necessary context
- Always-current property: when strategy updates, external views update automatically — no re-briefing
- Trust levels per relationship determining what additional context is shared

This is both a product differentiator and a GTM enabler: advisory firms receive structured context from their clients (instead of manual briefings), making Stratafy indispensable to the advisory relationship. Also strengthens the consumption-as-maintenance flywheel — external consumers stress-test the architecture, feeding discoveries back into the workbench.

Directly supports the "context layer becomes a network" thesis: value increases with number and diversity of consumers, including external ones.

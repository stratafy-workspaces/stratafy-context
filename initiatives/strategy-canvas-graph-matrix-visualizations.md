## Overview

Visual strategy exploration through multiple perspectives. Strategy Canvas provides an interactive tree, Strategy Graph renders network visualisations, and Strategy Matrix maps organisational and financial axes — enabling users to see their strategy from tree, network, and matrix views.

## Scope

- **Strategy Canvas**: Interactive tree with audience modes (leadership, team, board), focus mode, and health indicators
- **Strategy Graph**: Vue Flow network visualisation with edge rendering showing relationships between strategic entities
- **Strategy Matrix**: Organisational axis and finance axis views for cross-cutting analysis
- Multiple visual perspectives on the same underlying strategic data
- Interactive exploration with drill-down and filtering capabilities

## Expected Outcomes

- Enable leadership to shift between structural, relational, and analytical views of strategy
- Surface strategic gaps and imbalances through visual pattern recognition
- Support board presentations and stakeholder communication with appropriate audience views
- Provide intuitive navigation of complex strategic hierarchies

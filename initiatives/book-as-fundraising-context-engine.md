# Book as Fundraising Context Engine

## Description
Use "Strategy as Infrastructure" (the book) as the primary investor context-transfer tool. The approach: after an initial investor call, send the book manuscript + investor framing document with 5 AI analysis prompts. Investors feed materials to their AI tools, which surface the thesis, market timing, defensibility, and architecture in a way a 30-minute pitch cannot. This is simultaneously the pitch AND the proof — the context transfer problem the book describes is the problem the investor is experiencing, and the AI-assisted analysis is the solution in action.

Deliverables:
1. Finalised book manuscript (markdown, ready for PDF/print)
2. Investor Framing Document — one-pager with problem, evidence, category, and the ask
3. Five base prompts for AI-assisted opportunity analysis (investment thesis, category, market timing, architecture, red team)
4. Workflow: Initial call → send materials → follow-up after AI analysis → deep dive

Success signal: Investor references specific book concepts or AI analysis outputs in follow-up conversation.

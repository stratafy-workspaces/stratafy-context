## Overview

Migrate the existing /business-strategy-coaches/ directory to /network/coaches/ as part of the unified /network/ marketplace. Preserve all existing SEO value while establishing the new architecture.

## Scope

### URL Migration
- 301 redirects from all existing coach profile URLs to new /network/coaches/[location]/[slug] structure
- Preserve meta titles, descriptions, YML content, and structured data
- Validate all redirects with crawl testing

### SEO Preservation
- Transfer all existing domain authority to new URL structure
- Update internal links across the site
- Submit updated sitemap to search engines
- Monitor search rankings during transition period

### New Architecture
- /network/ landing page as SEO authority hub for both coaches and partners
- /network/coaches/ inheriting all existing coach directory content
- Shared filtering by region, industry, and methodology

### Timing Constraint
Only execute after /network/ architecture is confirmed and We Are 52 partner listing is ready to launch simultaneously. Launching /network/coaches/ alone without /network/partners/ loses the unified marketplace narrative.

## Expected Outcomes

- Zero SEO value loss during migration
- Unified /network/ hub with consolidated authority
- Simultaneous launch of coaches and partners directories
- Foundation for scalable marketplace growth

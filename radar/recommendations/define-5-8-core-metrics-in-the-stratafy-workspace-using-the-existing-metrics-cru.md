# Recommendation

> Define 5-8 core metrics in the Stratafy workspace using the existing metrics CRU

## Description
Define 5-8 core metrics in the Stratafy workspace using the existing metrics CRUD. Candidates: weekly commit velocity, content articles published (cumulative), MCP tool usage (from usage_events), active customer count, pipeline qualified prospects, workspace snapshot completeness score, strategy tree coverage (% of strategies with objectives). These metrics should be visible in the workspace dashboard and serve dual purpose: (1) dogfood the metrics layer proving it works, (2) create the data foundation for investor materials showing traction.

# Recommendation

> Before the Sequoia pitch window, define the pricing model as a strategy decision

## Description
Before the Sequoia pitch window, define the pricing model as a strategy decision — even if not implemented technically. Document: what unit you charge for (workspace, seat, agent, resolution level, consumption), what tiers exist, where STIR's R25k/month sits in that model, and what the ScaleUp commercial structure would be. This doesn't require billing infrastructure yet, but it needs to exist as a clear framework that can be communicated to investors. The pricing model also informs product architecture decisions (per-agent pricing changes how the Agent Registry works; resolution-based pricing changes how context packaging works).

# Stratafy ingestion from Notion/Google Drive is fast enough to neutralise sunk cost objection

## Description
The 'promotion not replacement' reframe only works if moving existing strategy content from Notion, Google Drive, or Claude context folders into Stratafy is genuinely fast and low-friction. If ingestion takes days of manual work, the sunk cost objection returns immediately and the reframe collapses. This is a product assumption as much as a sales one — the ingestion path must be demo-ready and feel like an upgrade, not a migration project.

## Assessment
- **Confidence**: hypothesis
- **Type**: operational
- **Impact if Wrong**: high
- **Status**: active

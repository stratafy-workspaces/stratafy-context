# Organizations will pay for strategic OS separate from PM/BI/collaboration

## Description
Companies will allocate budget specifically for a strategic operating system as a distinct line item, not bundled into existing PM, BI, or collaboration tool spend. This means willingness-to-pay exists at R25K-100K/month for strategic infrastructure. If wrong, Stratafy competes for existing tool budgets rather than creating new ones, making sales cycles longer and price pressure higher.

## Assessment
- **Confidence**: hypothesis
- **Type**: financial
- **Impact if Wrong**: critical
- **Status**: active

## Validation
- **Method**: Track willingness-to-pay signals from lighthouse customers. Did STIR allocate new budget or redirect existing tool budget? Do pipeline prospects frame Stratafy as replacing something or as a new investment? Target: 3+ customers paying from new budget allocation by Q3 2026.
- **Validated At**: Not yet validated
- **Evidence**: STIR paying R25K/month as a retainer — but this may be consulting budget, not tool budget. Need more data points to distinguish.

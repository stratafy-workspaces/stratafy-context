# Notion TCO exceeds Stratafy cost when manager time is included

## Description
As a Notion workspace grows in complexity — more agents, more databases, more automations, more team members, more strategy content — the time cost of the person managing it grows disproportionately. This is the hidden labour cost of the document model. Sam's 15 hours to build the current system is the initial build cost. But every strategy update, every new agent, every new team member onboarding, every Notion credit optimisation exercise, and every system audit adds to an ongoing maintenance burden that is difficult to see but real. The assumption is that this total cost — manager time × hourly cost + Notion credit spend + tool fragmentation overhead — is significantly higher than Stratafy's subscription cost, and that this can be demonstrated with a simple calculation. Sam's CRM-in-Notion example is instructive: she replaced a purpose-built CRM (ParkDrive) with a custom Notion build that requires ongoing maintenance. That decision trades upfront simplicity for long-term complexity — the same trade-off as using Notion for strategy instead of Stratafy.

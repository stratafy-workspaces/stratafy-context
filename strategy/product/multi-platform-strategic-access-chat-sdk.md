# Multi-Platform Strategic Access (Chat SDK)

## Description
Deploy Stratafy's strategic intelligence across every conversational surface via Chat SDK — transforming it from a destination product into ambient strategy infrastructure present at every decision point.

## Content
## Strategic Intent

**Question answered**: How do we make strategy present at the moment of decision, not locked in a separate app?

**Core thesis**: The biggest execution gap isn't that strategy doesn't exist — it's that strategy isn't present when decisions happen. Chat SDK enables a single codebase to project role-aware strategic intelligence into 7+ platforms, with rich cards, slash commands, thread subscriptions, and AI streaming.

## Why This Matters

This directly addresses the MCP demo accessibility risk (score: 9), reduces the three-tool-stack adoption friction, and operationalises the belief that "strategy consumption keeps strategy alive."

## Platform Strategy

A single codebase via Vercel's unified Chat SDK projects into every conversational surface where teams already work — Slack, Google Chat, Teams, Discord, Telegram, Linear, and GitHub. This makes strategic context ambient and accessible without requiring users to switch tools.

## Capabilities Per Platform

Rich cards for strategy summaries, slash commands for quick lookups, thread subscriptions for decision tracking, and AI streaming for real-time strategic conversations — all role-aware and context-filtered.

## Phasing

Google Chat + Telegram (live/testing) → Slack (next priority, richest adapter) → Teams (enterprise) → Discord + GitHub + Linear (ecosystem expansion).

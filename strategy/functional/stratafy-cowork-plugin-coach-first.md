# Stratafy Cowork Plugin (Coach-First)

## Description
Build and publish a unified "stratafy" Cowork plugin designed coach-first. Coaches are our primary GTM channel — they build their own strategies first, then onboard clients. By designing the plugin around coaching workflows, we automatically serve all four personas (coaches, strategy teams, employees, external advisors) because coaching workflows are a superset of company workflows. The plugin bundles 6 skills (strategy-foundations, workspace-builder, coaching-facilitation, execution-tracking, signal-intelligence, strategic-reporting) and 6 commands (/stratafy:setup-workspace, /stratafy:session-debrief, /stratafy:pulse, /stratafy:strategy-review, /stratafy:radar-scan, /stratafy:decision-brief) with our existing MCP server connector (135+ tools). Coaches install the plugin, connect their Stratafy API key, and Claude becomes their strategy workspace assistant. Low technical effort since our MCP server already exists; the work is primarily content authoring (skill markdown, command definitions). First strategy platform in the Claude plugin directory = category positioning advantage. GTM sequence: coach builds own workspace → onboards clients → employees and strategy teams adopt → advisors get invited.

# Strategy-to-Stratafy Team member linking: support named AI executive team members as strategy owners

## Description
The current data model supports a human lead_id and team_id on strategies. The question is whether to extend this to support a Stratafy Team member (named AI executive role) as a responsible executive per strategy. The linking unit should be the Team member role (e.g. Finance Director, CSIO, CTO) — not the plugin slug and not an agent instance ID — because multiple employees can use the same plugin, but the executive role is the accountable entity. This shapes how the Agent Registry initiative gets built.

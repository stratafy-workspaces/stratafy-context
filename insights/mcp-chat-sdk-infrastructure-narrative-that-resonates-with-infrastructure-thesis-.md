# MCP + Chat SDK = infrastructure narrative that resonates with infrastructure-thesis investors

## Summary
MCP for depth + Chat SDK for breadth creates a genuine infrastructure story comparable to Datadog/LaunchDarkly — the narrative Madrona and Sequoia back.

## Description
With MCP as the deep integration layer and Chat SDK as the broad distribution layer, Stratafy can now articulate a clear infrastructure narrative to investors: "We don't compete with where teams work — we embed into where they already are." This is the same pattern as Datadog (observability everywhere), LaunchDarkly (feature flags everywhere), or Segment (data collection everywhere). The investor pitch moves from "here's a strategy product" to "here's strategy infrastructure for the AI era that works across your entire communication stack." This narrative is particularly powerful for Madrona (who struggle with AI tool context across their portfolio) and Sequoia (who back infrastructure plays).

# AI advisory quality follows a context × lens specificity curve — Stratafy owns the context axis

## Summary
Grok reviews proved that AI advisory quality = context specificity × lens specificity. Generic AI tools can apply good lenses but lack organizational context. Advisory firms have good lenses and some context but limited depth. Stratafy uniquely owns the deep context axis, making it the infrastructure layer that makes all lenses more valuable.

## Description
When Grok was given Stratafy's methodology (domain-specific technical content) and asked for a Technical Advisor review, it produced specific, useful gaps — edge deployment considerations, governance readiness, vertical strategy. When given the same input and asked for a Startup Advisor review, it produced generic SV playbook advice indistinguishable from what it would say to any startup.

This reveals a pattern about AI-generated strategic advice: it's only as good as the intersection of (a) the specificity of the context provided and (b) the specificity of the lens applied. Generic context + generic lens = noise. Specific context + specific lens = signal.

Implications for Stratafy's market positioning:
- The advisory firm channel (EY, McKinsey, Scaling Up coaches) currently delivers high-quality lenses but with limited context depth (quarterly meetings, slide decks). Stratafy delivers deep context. The combination is more valuable than either alone.
- Generic AI tools (ChatGPT, Grok, Claude without Stratafy context) can produce Technical Advisor-quality reviews when given specific documents, but cannot produce operational reviews (CFO, COO, CHRO) without access to the actual operational data. This is Stratafy's defensible wedge.
- The "advisor in a box" pitch only works if the box contains the organization's full strategic context. Without it, you get STR-120. With it, you get something advisory boards can't match.

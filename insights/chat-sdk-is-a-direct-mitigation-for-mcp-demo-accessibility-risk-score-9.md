# Chat SDK is a direct mitigation for MCP demo accessibility risk (score 9)

## Summary
Chat SDK enables demos via Slack/Teams/Google Chat — eliminating the MCP setup barrier for investors and prospects. Directly mitigates top technical risk.

## Description
Chat SDK directly mitigates the highest-scored technical risk in the workspace: MCP-first architecture limits demo accessibility (risk score 9). With Chat SDK, Stratafy demos can run inside a Slack workspace, Google Chat space, or Teams channel — environments investors and prospects already use daily. No Claude Desktop setup, no MCP configuration, no technical prerequisites. An investor receives a Slack invite, @mentions the Stratafy bot, and sees strategic intelligence flowing in real-time with rich cards and streaming responses. This transforms the demo experience from "let me set up your Claude" to "check your Slack."

# Chat SDK inverts Stratafy's access model from pull to push

## Summary
Chat SDK enables push-based strategic access across 7+ platforms, complementing MCP's pull-based deep integration. Together they form the full infrastructure story.

## Description
Stratafy's current MCP-first architecture constrains demos and adoption to Claude Desktop setups. Chat SDK flips the access model entirely: strategic intelligence can be delivered via Slack channels, Google Chat spaces, Teams conversations, and Telegram bots — tools prospects and investors already have open. This doesn't replace MCP; it creates a complementary distribution layer. MCP remains the deep integration for power users, while Chat SDK becomes the broad reach layer for ambient strategic access. The combination is the full "strategy infrastructure" story.

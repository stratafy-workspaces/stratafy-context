# JSX cards enable native strategic UI rendering across all chat platforms

## Summary
Chat SDK's JSX cards render strategy dashboards, OKR progress, and risk alerts as native UI elements in each platform — no external links needed.

## Description
Chat SDK's JSX card system renders rich, interactive UI components natively on each platform — Slack Block Kit, Teams Adaptive Cards, Google Chat cards, Discord embeds, etc. This means Stratafy can render strategy dashboards, OKR progress bars, risk heat maps, decision status cards, and alignment scores directly inside chat platforms, not as links back to a web UI. This is a significant UX advantage: strategic intelligence appears as a native, first-class element of the conversation rather than an external redirect. Combined with action buttons (approve decision, escalate risk, acknowledge signal), it creates genuine in-flow strategic interaction.

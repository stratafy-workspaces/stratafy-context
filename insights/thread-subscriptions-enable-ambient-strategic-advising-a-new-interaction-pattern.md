# Thread subscriptions enable ambient strategic advising — a new interaction pattern

## Summary
Chat SDK thread subscriptions allow Stratafy to proactively inject strategic context into active work discussions — creating ambient strategic advising.

## Description
Chat SDK's thread subscription model maps perfectly to Stratafy's agent context packaging. When a team discusses a product decision in a Slack thread, the Stratafy bot can subscribe and inject relevant context: which initiative this relates to, the key assumptions at stake, related risks, and the strategic rationale. This creates a new category of interaction — ambient strategic advising — where strategic context appears proactively in the flow of work rather than requiring someone to seek it out. It validates the belief that "strategy consumption keeps strategy alive" by making consumption automatic and contextual.

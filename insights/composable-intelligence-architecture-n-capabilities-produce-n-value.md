# Composable intelligence architecture — N capabilities produce N² value

## Summary
Perspective Reviews reveal that Stratafy's intelligence capabilities (alignment scans, radar, perspective reviews, decisions, insights) are composable — each one enriches all others through shared context. 14 independent analytical channels all reading from and writing to the same workspace creates compound value and irreplicable switching cost.

## Description
The Perspective Reviews feature spec reveals something about Stratafy's product architecture that hasn't been explicitly named: Stratafy is becoming a platform where different intelligence capabilities compose on top of a shared context layer.

Current capabilities and how they relate:
- Foundation Alignment: checks strategy against identity (vertical)
- Strategic Coherence: checks strategies against each other (horizontal)  
- Radar: scans external environment through strategic lens (outside-in)
- Public Narrative: compares external perception against intent (inside-out)
- Perspective Reviews: applies functional expertise against full context (domain-specific)
- Organigram Alignment: checks structure against strategy (capacity)
- Decision Log: captures choice rationale against strategic context
- Insights: captures learning against strategic context

All of these are different analytical passes over the same underlying data. They're composable — you can run any combination and they enrich each other. A CFO perspective review that references a recent radar finding about market contraction is more valuable than either alone.

This is the "intelligence compound" effect: N capabilities produce N² value because each capability's output enriches every other capability's input.

At 8 perspective lenses + 4 alignment scans + insights + decisions, you have ~14 intelligence channels all reading from and writing to the same strategic context. After 6 months, the cross-references alone create switching cost that no competitor can replicate by copying features.

This should be named explicitly. It's not just "compounding intelligence" (which sounds abstract). It's "composable strategic intelligence" — independent analytical capabilities that multiply each other's value through shared context.

Pitch framing: "Every month, Stratafy runs your strategy through 14 independent analytical lenses. Each one makes every other one smarter. After 6 months, you have an intelligence infrastructure that no consultant, no AI tool, and no competitor can replicate."

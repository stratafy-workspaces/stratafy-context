# STIR's Level 1 B-BBEE + vertical integration is rare competitive moat

## Summary
Level 1 B-BBEE combined with equipment-to-distribution vertical integration is a rare combination in SA events. Key advantage over Gearhouse in procurement.

## Description
STIR Collective holds Level 1 B-BBEE status (highest possible) combined with full vertical integration from equipment ownership through studios, set design, content production, to distribution. This combination is rare in the SA events industry. Gearhouse Group, the primary competitor, has larger scale (35 years, 3 cities, 28 managed venues, 9 subsidiaries) but lacks STIR's content-to-distribution chain and B-BBEE advantage. A sector-specific BEE scorecard for technical production is being developed with DSAC, which could further advantage Level 1 contributors.

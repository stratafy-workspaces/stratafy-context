# HydraDB pricing sets a floor: generic context infra at $249–$5k/month validates Stratafy's pricing power

## Summary
Generic context infrastructure (no domain, no governance) prices at $249–$5k/month. Stratafy delivers domain-structured strategic context with scoring, role governance, alignment scanning, and semantic retrieval. Should price at or above HydraDB's range, not against general productivity SaaS benchmarks.

## Description
HydraDB charges $249/month (5 tenants, 1M tokens), $1,000/month (10 tenants, 100M tokens), and $5,000/month (unlimited, self-host). This is for generic context and memory infrastructure — no domain knowledge, no entity scoring, no role-based governance, no alignment scanning, no distillation pipeline, no strategic structure whatsoever. It stores and retrieves arbitrary context chunks.

Stratafy delivers all of that plus: typed strategic entities (risks with likelihood/impact scores, assumptions with confidence levels, decisions with type classification), role-aware context governance (7 roles × 11 entity types), distillation pipeline producing 4-depth AI-optimized summaries, alignment scanning across foundation/coherence/presence lenses, semantic retrieval with query-intent routing, and the full strategic operating system on top.

The pricing implication is significant: if the market pays $249–$5,000/month for the infrastructure layer alone, Stratafy — which is infrastructure plus domain application plus governance — should not be benchmarked against general productivity SaaS ($20–$50/user/month). It should be benchmarked against context infrastructure plus strategic advisory value combined.

Concrete action: review current pricing model against this benchmark. The Starter/Growth/Scale/Enterprise tiers and coach pricing should be stress-tested: are they capturing the infrastructure value, not just the SaaS application value? A company paying $5,000/month for HydraDB gets generic memory. A company paying the same for Stratafy gets strategic operating infrastructure with AI executives, alignment governance, and continuous context management. The delta should be reflected in price.

# Compound before you promote — SEO indexing decision reveals a reusable GTM timing pattern

## Summary
The decision to index methodology pages while keeping them hidden from nav is an instance of a broader pattern: start compounding loops early and promote when accumulated value is undeniable. Applies to SEO, content, product, data collection, and methodology — all feeding toward the April pitch window.

## Description
The SEO decision to add methodology pages to the sitemap while keeping them out of the nav (STR-116) represents a broader pattern worth formalizing: "compound before you promote." The principle applies across multiple dimensions:

- SEO: index pages weeks before they're nav-visible so they have ranking momentum when promoted
- Content: publish methodology articles (17 so far) to build authority before the pitch window
- Product: dogfood features internally before customer-facing launch
- Data: start collecting execution session data now (Daily Execution Session Logging) so there are 20+ data points for the Pieter meeting
- Methodology: build the co-working narrative into pages before making them the centerpiece of the pitch

In each case, the pattern is: start the compounding loop early, let it run in the background, then promote when the accumulated value is undeniable. This is the Cold-Start Moat principle applied to go-to-market timing. It's the opposite of the "launch day" mentality where everything goes live at once with zero accumulated value.

# Alignment scan confidence limited by strategy content depth — scans should consume initiatives and objectives too

## Summary
6 of 7 STIR scans rated "medium" confidence because strategy descriptions are sparse (1-2 sentences). Scans should consume the full strategy context: description + initiatives + objectives + metrics. Richer input = higher confidence = more valuable output.

## Description
The STIR alignment scans consistently flagged "lack of detailed strategic content" as a limitation on confidence. Every review scored "medium" confidence (except Group Integration at "high") because strategy descriptions are 1-2 sentences. The scans can only evaluate what's written. This reveals a product insight: strategy content depth directly affects scan quality. The platform should (a) show users how content depth affects their alignment confidence, (b) prompt users to enrich strategy descriptions before scanning, or (c) use initiative and objective data to supplement sparse strategy descriptions during scans. The current scan only reads the strategy description — it should also consider the initiatives and objectives underneath the strategy to build a richer picture.

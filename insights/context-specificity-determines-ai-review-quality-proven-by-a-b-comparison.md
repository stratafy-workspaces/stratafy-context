# Context specificity determines AI review quality — proven by A/B comparison

## Summary
Two Grok reviews with identical inputs but different lenses produced dramatically different quality. Technical Advisor: 5 specific gaps, 2 actionable. Startup Advisor: generic playbook, ~0 new signals. The variable is context specificity — proving that Perspective Reviews inside Stratafy (with full workspace context) will outperform any external AI review tool.

## Description
Two Grok reviews given the same input (methodology + glossary) but different lens prompts produced dramatically different quality output. The Technical Advisor review engaged deeply with actual architecture, producing 5 specific gaps — 2 actionable, 3 worth deferring. The Startup Advisor review produced generic "how to build a startup" advice that could apply to any pre-seed company.

The critical variable wasn't the AI model — it was context specificity. The Technical Advisor had domain-specific content (methodology architecture) to analyze. The Startup Advisor had the same input but the lens didn't engage with it — startup advice depends more on operational context (traction, funding, team) than on technical architecture.

This proves the Perspective Reviews product thesis: context injection is everything. The same lens (e.g., CFO) applied to generic context produces generic advice. Applied to Stratafy's full workspace (strategies, metrics, assumptions, risks, decisions, execution state), it would produce highly specific, actionable findings.

This is also a competitive moat argument: no external AI tool can replicate this because they don't have the structured strategic context. A founder using ChatGPT or Grok for a "CFO review" gets STR-120 (generic). A founder using Stratafy's Perspective Reviews gets something dramatically more useful because the system knows their actual numbers, assumptions, and execution state.

Evidence:
- STR-119: Technical Advisor — high signal, specific gaps, productive 2/3/2 triage
- STR-120: Startup Advisor — low signal, generic patterns, ~0 new signals
- Same AI, same input documents, different context engagement = different quality
